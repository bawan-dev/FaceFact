# FaceFact

FaceFact is an explainable facial grooming and skincare recommendation system. It analyzes user-provided face photos and returns low-risk, category-level suggestions with clear explanations and uncertainty.

## Non-goals
- Not an attractiveness scoring app
- Not a medical diagnosis tool
- Not a replacement for professional care

## Safety
- Observations are based on visible features and can be wrong
- Recommendations are low-risk and category-level only
- User allergies and exclusions are always respected

## Status
- Stage 1: project scaffolding

## Quick start
1. Create and activate a virtual environment
2. Install dependencies from requirements.txt
3. Run the Streamlit app (coming in MVP stage)
